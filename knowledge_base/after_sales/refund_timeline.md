# 退款时效说明

## 退款到账时间

| 支付方式 | 退款到账时间 | 备注 |
|---------|------------|------|
| 微信支付 | 1-3 个工作日 | 退回微信零钱或银行卡 |
| 支付宝 | 1-3 个工作日 | 退回支付宝余额或银行卡 |
| 花呗分期 | 1-3 个工作日 | 已出账单需自行还款 |
| 银联云闪付 | 3-5 个工作日 | 退回原银行卡 |
| Apple Pay | 3-5 个工作日 | 退回原银行卡 |

## 退款流程节点

1. **申请提交** — 用户提交退款/退货申请
2. **审核通过** — 客服审核通过（仅退货、取消订单）
3. **退货验收** — 商家收到退货并确认完好（仅退货退款）
4. **退款发起** — 系统向支付渠道发起退款
5. **退款到账** — 款项到达用户账户（上述时效从此节点开始计算）

## 各场景退款节点

### 仅退款（未发货取消订单）
- 申请 → 审核 → 退款到账
- 总耗时：通常 2 小时内完成

### 退货退款
- 申请 → 审核 → 用户寄回 → 商家验收 → 退款到账
- 总耗时：5-15 天（取决于物流时间）

### 质量问题退款
- 申请 → 商家确认 → 用户寄回（商家承担运费）→ 验收 → 退款到账
- 总耗时：5-15 天

## 退款金额说明

| 项目 | 说明 |
|------|------|
| 商品金额 | 全额退款 |
| 运费 | 质量问题由商家承担；无理由退货由买家承担 |
| 优惠券 | 按比例退回（未过期可继续使用） |
| 积分 | 按比例退回 |
| 花呗手续费 | 已出账单不免手续费；未出账单免手续费 |

## 超时处理

- 退货签收后超过 3 个工作日未退款，系统自动升级为加急处理
- 超过 7 个工作日未退款，可拨打 400-888-6666 投诉
- 可申请平台介入的时间：发起退货后 15 天
