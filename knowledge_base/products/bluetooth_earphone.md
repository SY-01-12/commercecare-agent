# 无线蓝牙耳机 Pro 商品说明

## 产品概述

无线蓝牙耳机 Pro（PROD-101）是声悦品牌旗舰级真无线降噪耳机，适用于日常通勤、运动和商务通话场景。

## 详细规格

| 参数 | 详情 |
|------|------|
| 品牌 | 声悦 |
| 型号 | Pro |
| 蓝牙版本 | 5.3 |
| 降噪 | 主动降噪 ANC，降噪深度 -42dB |
| 续航 | 单次 8 小时，含充电仓总计 36 小时 |
| 防水等级 | IPX5（防汗防雨） |
| 充电接口 | USB-C，支持无线充电 |
| 音频编码 | AAC / SBC / LDAC |
| 重量 | 单耳 5.2g，充电仓 48g |

## 包装清单

- 耳机 × 2（左右耳各一）
- 充电仓 × 1
- USB-C 充电线 × 1
- 硅胶耳塞 × 3 对（S/M/L）
- 说明书 × 1

## 保修信息

- 保修期限：12 个月
- 保修范围：非人为质量问题免费维修或换新
- 不保修范围：进水损坏、跌落碎裂、私自拆卸、使用非原装充电器导致的损坏

## 使用注意事项

- 首次使用请充满电（约 1.5 小时）
- 充电仓指示灯：绿色常亮 = 满电，红色闪烁 = 电量低
- 支持单耳独立使用
- 触控操作：双击播放/暂停，长按切换降噪模式
- 恢复出厂设置：长按充电仓按键 10 秒
